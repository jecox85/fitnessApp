Bodyweight Fitness Recommended Routine

Congratulations on choosing the recommended routine for your fitness journey. This routine was designed to use calisthenics to achieve a full-body workout.

Workouts are meant to be done 3 times a week with a minimum of one day between each exercise.

You will need access to a few items for this routine, although suitable alternatives can be found:

  • A place to do rows.
  • A place to do pull-ups
  • Parallel bars

There are nine exercises total spit up into pairs and one set of triplets.

Rest 90 seconds or up to 3 minutes if needed between each set of pairs, and 60 seconds between the triplet exercises.

Check out the about page for more information on this program and how progressions work.

Good luck and have fun!
